1. Если еще не инсталлирован, то установите Питон с официального сайта: https://www.python.org/downloads/ 
Для Windows 7/Windows Server последняя поддерживаемая версия 3.8.8: https://www.python.org/ftp/python/3.8.8/python-3.8.8-amd64.exe
В процессе установки нужно --> ОБЯЗАТЕЛЬНО <-- поставить галочку Add Python to Path/to environment...

2. Распакуйте архив в папку OlimpCompare (нажмите правой кнопкой по сохраненному архиву и выберите  Извлечь в "OlimpCompare").
3. Перейдите в Проводнике в папку из п.2 (куда распаковали содержимое архива).
4. Установите нужные библиотеки: для этого один раз запустите файл !!!_install_libs.bat. 
Если все правильно сделали, то почти в самом конце напишет Successfully installed.
5. В файл config.ini при необходимости внесите ваши данные (редактировать можно обычным текстовым редактором, например через Блокнот: запустите Блокнот и перетащите файл в его окно). 
6. Сохраните config.ini.
7. Для запуска файл !!!_start.bat.
